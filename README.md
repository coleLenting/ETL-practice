## Simple ETL Project

A Python-based ETL (Extract, Transform, Load) pipeline that processes data from multiple sources <b>CSV, JSON, and XML</b> transforms it, and loads it into a consolidated CSV file.<br>
### `Features:`
<li>Extracts structured data from CSV, JSON, and XML files.<br>
<li>Transforms data (e.g., rounds numeric values).<br>
<li>Loads transformed data into a single CSV output.<br>
<li>Logs progress for each ETL phase (Extraction → Transformation → Loading).<br><br>

## `Tech Stack:`
<li>Python 3<br>
<li>pandas<br>
<li>xml.etree.ElementTree<br>
<li>glob<br>
